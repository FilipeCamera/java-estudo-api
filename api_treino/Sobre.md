# Desenvolvimento Full Stack 

Vamos desenvolver uma aplicação de criação de treino de academia, onde o Personal ele cria o treino do aluno anual e seleciona o prazo de cada treino, 7 - 15 - 30 dias, onde ao chegar no prazo o treino é trocado por outro pré determinado pelo Personal.
(O personal ele irá criar o treino completo do aluno e estipular os prazos de cada treino.)

- Aplicar os princípios do SOLID e Arquitetura Limpa
- Testes unitários
- Testes e2e
 
### Linguagens: JAVA, Typescript

### Back: SpringBoot
### Front: Vue.js

S - Single Responsibility Principle
O - Open Closed Principle
L - Liskov Substitution Principle
I - Interface Segregation Principle
D - Dependency Inversion Principle
